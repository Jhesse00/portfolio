# Johanne Hesse Portfolio

A modern portfolio site for showcasing cybersecurity, development, and systems projects. The site highlights featured work, technical skills, resume access, and contact options in a responsive light/dark theme.

## Features

- Responsive portfolio layout for mobile, tablet, and desktop
- Light and dark theme support
- Featured project cards with categories, technology tags, and GitHub links
- Resume section with downloadable PDF support
- Contact form powered by EmailJS
- Framer Motion animations for smooth section transitions

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Express
- esbuild
- EmailJS

## Setup

Install dependencies:

```bash
npm install
```

Start the local server:

```bash
npm start
```

Open the site at:

```text
http://localhost:4000
```

Build production assets:

```bash
npm run build
```

On Windows PowerShell, use `npm.cmd` if script execution policy blocks `npm`:

```powershell
npm.cmd install
npm.cmd run build
```

## Project Structure

```text
src/components/          Reusable page sections and UI components
src/components/projects/ Project-specific display components
src/data/                Portfolio content data
src/lib/                 Shared utilities
public/                  Static assets and built output
scripts/                 Build scripts
```
